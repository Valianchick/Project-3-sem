# Покупка комрьютеров с учетом заданных параметров 

## Описание:
Программа предназначена для выбора ноутбука исходя из данных параметров. 

## Задачи: 
1. Генерация базы данных. 
2. Реализация функционала. 
3. Создание интерфейса.
